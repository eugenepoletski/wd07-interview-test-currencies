Конвертер и калькулятор валют

Обязательно использовать react.js и redux
Для UI использовать Bootstrap (можно 4.0beta)

Приложение должно уметь следующее:

1. Переодически запрашивать и обновлять локально хранящиеся курсы валют (историческую информацию не накапливать, показывать только последние курсы)
Данные можно получать с сервиса Центрального банка РФ http://cbr.ru/ или https://finance.yahoo.com/

2. UI должен иметь следующие закладки (Tab) - Конвертер, Курсы
3. Tab "Курсы" должен отображать таблицу с курсами (как вариант примерно такой http://cbr.ru/currency_base/daily.aspx?date_req=07.09.2017) и показывать последнюю дату и время обновления
4. Tab "Конвертер" должен позволять выбрать базовую валюту и любое количество дополнительных
5. Пользователь должен иметь возможность ввести любую сумму в базовой валюте, по потере фокуса должен быть выполнен расчет для остальных выбранных валют.
6. UI должен корректно отображаться как на широком экране так и на телефоне.
